# Simple Multi-Page Website

A responsive multi-page website built with **HTML5** and **CSS3**. The project demonstrates basic website structure, navigation, responsive layouts, forms, and embedded media without using JavaScript.

## Features

- Multi-page navigation
- Responsive landing page
- Fixed navigation bar
- Product cards
- Login form
- Embedded YouTube video
- Modern styling with Google Fonts

## Pages

- **Home** – Hero section with call-to-action
- **About** – Embedded YouTube video
- **Products** – Three product cards
- **Login** – Simple login form

## Technologies

- HTML5
- CSS3

## Project Structure

```
.
├── index.html
├── about.html
├── products.html
├── login.html
├── style.css

```

## Getting Started

1. Clone the repository.
2. Open `index.html` in your browser or run a local web server.
