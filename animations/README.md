# HTML & CSS Animations (Day 2)

A small collection of CSS animations built as part of a daily web development challenge. This project explores keyframes, transitions, transforms, and animation properties using pure HTML and CSS.

## Features

- Hover-triggered animation
- Infinite loading animation
- CSS keyframes
- 2D & 3D transforms
- Pure HTML & CSS (no JavaScript)

## Technologies

- HTML5
- CSS3

## Run

Open `index.html` in your browser.

## About

This is **Day 2** of a series where each day is a new standalone web development project to practice modern frontend development.
